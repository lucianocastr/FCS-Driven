Isolation improvement tips


1. Verify correct installation and performance of all cables and passive devices.
2. Check donor antenna positioning/azimuth. Donor antenna should be installed as close to the outer edge of the building's rooftop as possible and pointing away from the building.
3. Increase the height of the donor antenna. Vertical separation can help increase isolation.
4. Locate the indoor antenna(s) causing low isolation. Attenuate or remove accordingly.
5. Replace donor antenna with a higher isolation donor antenna.

WARNING: IF DURING THIS STEP, THE DONOR ANTENNA OR RELATED PASSIVE DEVICES NEEDED TO BE MODIFIED FROM THEIR ORIGINAL INSTALLATION, THEN THE USER MUST EXECUTE STEPS 2, 3 AND 4 AGAIN IN ORDER TO VERIFY THE CORRECT DOWNLINK POWER RECEPTION.




1. The Clear Alarm button allows the user to mute/cancel the Oscillation Detection Alarm.
2. Oscillation Time Threshold defines the abnormal AGC actuation time limit which will be considered an oscillation. This time limit should be longer than the maximum call duration of a portable radio in the system (default/recommended value is 60 seconds).
3. Action After Alarm sets the action that the BDA performs once an oscillation is detected. Actions can be:
  • Automatic Shutdown: The BDA removes the oscillation condition by temporarily disabling the power amplifiers. The BDA will then attempt to turn on power amplifiers and check for oscillation condition after the retry time period. If the oscillation detection alarm remains after 4 retry attempts, the power amplifiers will remain disabled until the user corrects the issue and turns the power amplifiers back on in the user interface.
  • Run Isolation Measurement:The BDA will automatically measure the isolation once the oscillation condition is detected. The BDA will then reduce the gain accordingly to adjust for the new isolation condition. If the isolation has become too low such that gain cannot be reduced enough to prevent oscillation, then the BDA will shut down the power amplifiers.
4. Retry timer after auto PA Off: This timer is used once the Power Amplifiers are shut down due to oscillation detection. The user can set the retry timer to turn on the PAs and check the oscillation condition. The BDA will attempt 4 retries before permanently shutting the power amplifiers off.


1. This option allows to switch off automatically the uplink power amplifier in order to reduce Uplink noise emission.
2. Timer defines the period of absence of uplink incoming signal that BDA uses to automatically switch of uplink PA.
3. Configuring timer to zero, the feature is disabled

Setup 700 MHz Band?



Would you like to use Narrow Band or Adj.BW filters?









  • Narrowband: Narrowband filters are used to provide channelization of system channels. Each channel is programmed into the BDA. Using narrowband filters provides key performance differentiators such as interference rejection, DL/UL Squelch per channel/timeslot and DL/UL AGC per channel/timeslot. These additional characteristics provide the most stable coverage, the best noise performance and provide near-far communications preservation.

  • Adj. BW: Adjustable Bandwidth filters are used when channelization is not desired. The user will set the bandwidth of the filter to encompass all required system channels. Adjustable bandwidth filters are wideband and will allow any frequency within the bandwidth to be amplified. Channel output power will vary depending on the traffic within the filter bandwidth.

DL RSSI improvement tips


1. Check to ensure donor antenna alignment/azimuth is correct and optimized.
2. Check cables & connections between donor antenna and the BDA to ensure correct performance.
3. Replace donor antenna with a higher gain donor antenna.

DL AGC Broadband warning


DL Broadband AGC will activate due to strong in-band or out-of-band signals at the input of the BDA. This could cause gain reduction across the entire passband of the BDA. In general, the composite power at the input of the BDA should not exceed -35dBm to prevent broadband AGC activation. If broadband AGC level is excessive, external attenuation and/or additional notch filtering could be required. The spectrum analyzer tool can be used to view signals across the BDA passband to help identify potential interference that would cause the Broadband AGC to activate.
